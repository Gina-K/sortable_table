# Задание 3. Таблица с постраничной навигацией
Сверстать простую таблицу с постраничной навигацией, используя предоставленный данные. Количество элементов на странице: 3.

## Требования
1. Реализовать переключение страниц.

2. Реализовать сортировку элементов таблицы по названию и дате (можно просто по нажатию на
   заголовок).
   
## Стэк
- jQuery
- HTML5
- CSS3


Посмотреть [работающую версию](https://gina-k.github.io/sortable_table/)
